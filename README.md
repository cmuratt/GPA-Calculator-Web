# GPA Calculator — Mobile PWA

> Developed by **Murat Can IŞIK**

Java Swing masaüstü uygulamasından dönüştürülmüş, mobil uyumlu Progressive Web App (PWA) versiyonu.

## Özellikler

- 📱 **Mobil uyumlu** — telefon, tablet ve masaüstünde çalışır
- 🏠 **Ana ekrana eklenebilir** — native uygulama gibi açılır
- ✈️ **Offline çalışır** — internet olmadan da kullanılabilir
- 💾 **Otomatik kaydetme** — verileriniz tarayıcıda güvenle saklanır
- 🌙 **Dark tema** — göz dostu koyu tasarım

## Modlar

### 1. Saved Profile
Gerçek akademik kaydınız için. Dönemler oluşturun, dersler ekleyin. Program verilerinizi otomatik olarak kaydeder.

### 2. Guest Mode
Hızlı "ya şöyle olsaydı?" senaryoları için. Kayıtlı profilinizi bozmadan yeni not senaryolarını test edin.

## Nasıl Kullanılır

1. Tarayıcıda `index.html` dosyasını açın
2. Veya bir web sunucusu ile servis edin:
   ```bash
   npx serve .
   ```
3. Telefonunuzda "Ana Ekrana Ekle" seçeneğini kullanarak PWA olarak yükleyin

## Teknolojiler

- HTML5 / CSS3 / JavaScript (Vanilla)
- Progressive Web App (PWA)
- Service Worker (Offline desteği)
- localStorage (Veri saklama)
